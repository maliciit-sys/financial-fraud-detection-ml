# Detection of Financial crimes leveraging Artificial Intelligence: Early warning and Pre-fraud detection

## Project Overview
Detection of financial crimes using PyTorch neural networks for early warning and pre-fraud detection.

## Dataset
- Total transactions: 5,000,000
- Fraudulent: 179,553 (3.59%)
- Non-Fraudulent: 4,820,447 (96.41%)
- Features: 13 (after preprocessing)

## Technology Stack
- Python 3.x
- PyTorch
- Pandas, NumPy, Scikit-learn
